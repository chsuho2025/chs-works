PROBLEM

단체 안무를 기록하는 데 초점을 두고 카메라를 고정해 촬영했더니, 안무는 담겼지만 시청자가 반응할 와우 포인트가 부족했습니다.

APPROACH

안무를 외울 정도로 익힌 뒤 킬링파트를 먼저 정했습니다. 브릿지 파트는 핸디캠을 활용한 역동적인 샷으로 설계했고, 후반에 적용할 색감과 효과를 미리 떠올린 상태에서 그에 맞는 조명을 설치했습니다.

RESULT

촬영은 단순히 대상을 기록하는 단계가 아니라, 연출 의도를 반영하는 단계라는 작업 방식을 정착시켰습니다.

PROBLEM

세 팀이 각자 수강신청하는 모습을 따로 촬영해 한 편으로 편집하는 방식이었습니다. 처음에는 시간순으로 이어 붙였더니, 영상이 촬영본 나열에 가까워 흐름이 만들어지지 않았습니다.

APPROACH

RESULT

리얼리티와 예능형 콘텐츠는 촬영본 자체가 아니라 흐름을 어떻게 설계하느냐에 따라 결과물 완성도가 달라진다는 점을 배웠습니다.

역할  1개 팀 촬영 + 전체 편집  ·  업로드  학과 유튜브 채널

Case Study 01 · 검수 프로세스 개선

레이어별 분리 점검 후 통합 확인

PROBLEM

완성된 영상을 통째로 재생하며 감각으로 검수하는 방식이었습니다. 영상에는 레이어가 많이 쌓여 있었고, 외부에서 받아온 정보도 있어 통합본 한 번 보는 것만으로 모든 요소를 점검하기 어려웠습니다.

ANALYSIS

직접 제작하지 않은 레이어가 다수 포함되어 있는 만큼, 책임지고 점검하려면 레이어 하나하나를 따로 살펴봐야 한다고 판단했습니다.

SOLUTION

비디오, 오디오, 자막, BI를 각각 분리해 한 레이어씩 따로 점검한 뒤, 마지막에 모든 레이어를 합쳐 어색한 지점이 없는지 다시 확인하는 방식으로 검수 흐름을 변경했습니다.

RESULT

이 방식을 정착시킨 이후 발행 영상에서 발생한 오류는 0건, 발행 전 시사 단계 오류는 59편 중 1건이었습니다. 인턴 동료들에게 문서로 공유해 팀의 공식 검수 체계로 반영되었습니다.

본 작업은 멈추지 않음

PROBLEM

납품 영상 개수가 갑자기 늘면서 2주 안에 39편의 숏폼을 제작해야 했고, 평소 대비 40% 늘어난 근무시간이 필요했습니다.

ANALYSIS

가장 큰 변수는 작업 속도 자체가 아니라 혼자 결정할 수 없는 변수들이었습니다. 심의에 걸릴 가능성이 모호한 장면, CP사와 협의해야 하는 각색 사항이 생기면 작업이 멈추고 답을 기다려야 했습니다.

SOLUTION

그때그때 묻는 방식 대신 직접 공유용 시트를 만들어 변수가 발생한 장면, 결정이 필요한 사항과 그 이유를 정리해 한 번에 공유하는 체계를 만들었습니다.

RESULT

2주 안에 39편을 모두 마감했습니다. 작업량과 속도가 동시에 요구되는 상황에서 핵심은 본인의 속도가 아니라, 통제 밖에 있는 변수를 어떻게 관리하느냐에 달려 있다는 점을 배웠습니다.

마감  39편 / 2주 · 100% 발행 완료
